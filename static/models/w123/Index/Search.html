These Index files do not presently have a search engine built in. We are working on one. In the meantime, there is a way for you to add the search function IF

  1. you understand the rudiments of adding already-written html code to page,
  2. you have a connection to the internet (perferrably persistant),
  3. you have personal web sharing enabled on your computer, and
  4. you are willing to subscribe to a free search engine provider like www.atomz.com.
